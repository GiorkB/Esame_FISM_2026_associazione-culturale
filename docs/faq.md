# Domande Frequenti

## Il sito funziona senza internet?
No, è necessaria una connessione a internet per la corretta visualizzazione e funzionamento del sito.

## Perché i dati non compaiono?
- Controllare la connessione a internet.
- Controllare che nei file HTML sia presente il collegamento a script.js.

## Come modifico i contenuti?
I contenuti sono modificabili accedendo ai file data.json e script.js.

## Dove si trova il file dei dati?
I dati sono contenuti nel file data.json.

## A cosa serve il file script.js?
Il file script.js serve ad accedere ai dati di data.json e presentarli nella maniera corretta nelle pagine del sito.
