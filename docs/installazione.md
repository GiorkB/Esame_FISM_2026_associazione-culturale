## Guida all'installazione

Dopo aver aperto il terminale, clonare la repository:
```bash
git clone https://github.com/GiorkB/Esame_FISM_2026_associazione-culturale.git
```
Posizionarsi nella cartella della repository:
```bash
cd Esame_FISM_2026_associazione-culturale
```
Poi eseguire il comando:

```bash
python -m http.server 8000
```

Poi aprire nel browser:

```text
http://localhost:8000
```
